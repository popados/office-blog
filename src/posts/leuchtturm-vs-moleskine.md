Both notebooks cost roughly the same. Both are considered "premium." Both show up on every desk setup inspiration post you'll find online. But after 60 days of using each one as my primary daily notebook, they are *not* the same product.

Here's the breakdown.

## The Basics

| Feature | Leuchtturm1917 A5 | Moleskine Classic L |
| ------- | ----------------- | ------------------- |
| Pages | 249 (numbered) | 240 (unnumbered) |
| Paper weight | 80 gsm | 70 gsm |
| Dot grid available | Yes | Yes |
| Index | Yes (pre-printed) | No |
| Page numbering | Yes | No |
| Ribbon bookmarks | 2 | 1 |
| Price | ~$23 | ~$21 |

## Paper Quality

This is where the gap is most obvious. Leuchtturm uses 80 gsm paper; Moleskine uses 70 gsm. That 10 gsm difference is enough to matter.

With fine gel pens (Pilot G2 0.38mm, Uni-ball One 0.38mm), the Moleskine shows ghosting — you can see the previous page through the paper. The Leuchtturm does not. With fountain pens, the Moleskine bleeds at anything wetter than a fine nib. The Leuchtturm handles medium nibs without issue.

If you write exclusively with ballpoints, you probably won't notice. For gel and fountain pen users, Leuchtturm is the clear choice.

## Usability

The Leuchtturm's numbered pages and built-in index table make it genuinely functional as a reference notebook. You can write "meeting notes p.34" in the index and find it later. Moleskine gives you nothing — no page numbers, no index.

Two ribbon bookmarks vs one sounds minor. It isn't. One bookmark for the current page, one for a reference page you return to often. Once you have two, you won't want to go back.

## The Feel

Moleskine wins on cover feel. The slightly softer cover and rounded corners give it a more refined look and better pocket carry. Leuchtturm's cover is stiffer and more utilitarian.

If the notebook lives on a desk: Leuchtturm. If it lives in a jacket pocket: Moleskine.

## The Verdict

**Leuchtturm1917** is the better notebook for desk use, daily journaling, planning, and anyone who uses fountain pens or fine gel pens. The page quality and organizational features make it worth the small price difference.

**Moleskine** wins if portability and aesthetics matter more than paper quality — or if you're buying for a gift and the recipient is brand-conscious.

For my daily workflow, the Leuchtturm replaced the Moleskine by week three and hasn't left the desk since.
